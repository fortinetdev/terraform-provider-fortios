# fortios_system_replacemsggroup
Configure replacement message groups.

## Example Usage

```hcl
resource "fortios_system_replacemsggroup" "trname" {
  comment    = "sgh"
  group_type = "utm"
  name       = "1"
}
```

## Argument Reference

* `name` - Group name.
* `comment` - Comment.
* `group_type` - (Required) Group type.
